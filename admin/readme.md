# GoWAF 网关管理系统功能清单

## 一、仪表盘模块

### 模块名称：仪表盘
**功能描述：**
- 系统总览：展示系统运行状态、关键指标和统计数据
- 实时监控：显示实时访问量、拦截情况、系统负载等核心指标
- 数据可视化：通过图表展示访问趋势、拦截统计、IP分布等信息
- 快捷入口：提供各功能模块的快速访问入口

---

## 二、IP地址管控模块

### 模块名称：黑名单管理
**功能描述：**
- 黑名单查询：支持按IP地址、类型（单个IP/网段）、状态、过期状态等条件筛选查询
- 黑名单列表：分页展示黑名单记录，包含IP地址、类型、来源、状态、过期时间、创建时间、创建人等信息
- 添加黑名单：支持添加单个IP或网段到黑名单，可设置来源、状态、过期时间和备注
- 编辑黑名单：支持修改黑名单的状态、过期时间和备注信息
- 删除黑名单：支持删除黑名单记录
- 过期时间管理：支持永久、预设时间（1天、7天、1个月）和自定义过期时间
- 操作日志：自动记录所有增删改操作到操作日志

### 模块名称：IP白名单管理
**功能描述：**
- 白名单查询：支持按IP地址、类型（单个IP/网段）、状态、过期状态等条件筛选查询
- 白名单列表：分页展示白名单记录，包含IP地址、类型、来源、状态、过期时间、创建时间、创建人等信息
- 添加白名单：支持添加单个IP或网段到白名单，可设置来源、状态、过期时间和备注
- 编辑白名单：支持修改白名单的状态、过期时间和备注信息
- 删除白名单：支持删除白名单记录
- 过期时间管理：支持永久、预设时间（1天、7天、1个月）和自定义过期时间
- 操作日志：自动记录所有增删改操作到操作日志

---

## 三、接口限流模块

### 模块名称：IP限流
**功能描述：**
- 限流配置查询：支持查询IP限流的全局配置
- 限流配置管理：支持配置IP限流的开启状态、时间窗口、请求次数限制、封禁时间
- 配置说明：展示限流配置的各项参数含义和使用说明
- 操作日志：自动记录所有配置修改操作到操作日志

### 模块名称：路由全局限流
**功能描述：**
- 全局限流配置查询：支持查询路由全局限流的配置信息
- 全局限流配置管理：支持配置路由全局限流的开启状态、时间窗口、请求次数限制
- 配置类型：配置类型为 url_global_limit
- 配置格式：content 格式为 {"open":true,"check_time":10,"check_num":100}
- 操作日志：自动记录所有配置修改操作到操作日志

### 模块名称：路由自定义设置
**功能描述：**
- 路由列表查询：支持按路由地址、类型、租户配置开启状态、IP配置开启状态、会员配置开启状态等条件筛选查询
- 路由列表展示：分页展示路由配置，包含ID、路由地址、类型、租户配置、IP配置、会员配置、更新时间等信息
- 类型区分：支持详细地址和通配符两种类型，用不同颜色徽章区分
- 添加路由配置：支持添加新的路由配置，可设置类型、路由地址、租户配置、IP配置、会员配置
- 编辑路由配置：支持修改路由配置的各项参数
- 删除路由配置：支持删除路由配置记录
- 配置管理：
  - 租户配置：支持配置开启状态、时间窗口、请求次数限制、封禁时间
  - IP配置：支持配置开启状态、时间窗口、请求次数限制、封禁时间
  - 会员配置：支持配置开启状态、时间窗口、请求次数限制、封禁时间
- 配置格式：各配置格式为 {"check_time":10,"check_num":100,"block_time":0}
- 操作日志：自动记录所有增删改操作到操作日志

---

## 四、访问设置模块

### 模块名称：Agent设置
**功能描述：**
- Agent配置查询：支持查询Agent的配置信息
- Agent配置管理：支持配置Agent的各项参数和规则
- 规则管理：支持添加、编辑、删除Agent规则
- 规则配置：支持配置规则的名称、类型、匹配条件、执行动作等参数
- 操作日志：自动记录所有配置修改操作到操作日志

---

## 五、系统管理模块

### 模块名称：账号管理
**功能描述：**
- 账号查询：支持按用户名、状态等条件筛选查询
- 账号列表：分页展示管理员账号，包含ID、用户名、真实姓名、手机号、邮箱、状态、创建时间等信息
- 添加账号：支持创建新的管理员账号，需设置用户名、密码、真实姓名、手机号、邮箱、角色、状态等
- 编辑账号：支持修改账号的真实姓名、手机号、邮箱、角色、状态等信息
- 删除账号：支持删除管理员账号
- 锁定/解锁：支持锁定和解锁账号
- 角色管理：支持管理员、操作员、查看者三种角色
- 操作日志：自动记录所有增删改操作到操作日志

### 模块名称：操作日志
**功能描述：**
- 日志查询：支持按操作人、操作模块、操作类型、操作结果、时间范围等条件筛选查询
- 日志列表：分页展示操作日志，包含ID、操作人、操作模块、操作类型、操作IP、操作时间、操作结果等信息
- 日志详情：支持查看操作日志的详细信息，包含操作内容（JSON格式）、错误信息、请求ID等
- 模糊查询：支持操作人、操作模块、操作类型的模糊查询
- 时间范围查询：支持按创建时间范围查询日志
- 操作类型：包含新增、修改、删除、锁定、解锁等操作
- 操作结果：包含成功（1）和失败（2）两种状态

### 模块名称：系统设置
**功能描述：**
- 设置查询：支持按设置类型筛选查询
- 设置列表：分页展示系统设置，包含ID、类型、内容、更新时间、更新人等信息
- 添加设置：支持添加新的系统设置，需设置类型和内容
- 编辑设置：支持修改设置的内容
- 删除设置：支持删除系统设置
- 设置类型：支持多种设置类型，包括IP限流、路由全局限流等配置
- 配置格式：content 字段支持JSON格式的配置数据
- 操作日志：自动记录所有增删改操作到操作日志

---

## 六、公共功能

### 模块名称：登录认证
**功能描述：**
- 用户登录：支持管理员用户名和密码登录
- 登录验证：验证用户名和密码的正确性
- 会话管理：管理用户登录会话和状态
- 退出登录：支持用户安全退出系统

### 模块名称：操作日志记录
**功能描述：**
- 日志记录：为所有模块的增删改操作自动记录操作日志
- 日志内容：包含操作人、操作类型、操作模块、操作内容、操作IP、操作结果、错误信息、请求ID等
- 日志存储：将日志数据存储到数据库表中
- 日志查询：提供操作日志的查询接口和页面

### 模块名称：分页功能
**功能描述：**
- 分页展示：所有列表页面都支持分页展示数据
- 分页控制：支持上一页、下一页、跳转到指定页等操作
- 每页条数：支持自定义每页显示的条数（10、20、50、100等）
- 总数显示：显示数据总数和当前页码信息

### 模块名称：数据验证
**功能描述：**
- 参数验证：对所有接口请求参数进行格式和类型验证
- 必填验证：验证必填参数是否提供
- 范围验证：验证参数值是否在允许的范围内
- 长度验证：验证字符串参数的长度限制
- 枚举验证：验证枚举类型参数的值是否合法

### 模块名称：权限管理
**功能描述：**
- 角色权限：支持管理员、操作员、查看者三种角色
- 权限控制：根据用户角色控制页面和功能的访问权限
- 操作权限：控制用户对数据的增删改查权限

- 响应式设计：支持不同屏幕尺寸的响应式布局

## 功能清单说明

本功能清单基于 GoWAF 网关管理系统的实际接口功能和页面功能整理，涵盖了系统的所有核心功能模块。每个模块都包含详细的功能描述，说明了该模块提供的主要功能和特性。

系统采用前后端分离架构，前端使用 HTML/CSS/JavaScript 开发，后端使用 Lua 语言开发，数据存储使用 MySQL 数据库。系统支持多用户、多角色的权限管理，具备完善的操作日志记录和审计功能。
