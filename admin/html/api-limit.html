限流配置说明
时间窗口

统计请求次数的时间范围，如1秒、10秒、60秒等

次数限制

在时间窗口内允许的最大请求次数

锁定时间

超过限制后，该IP被锁定的时间长度

优先级

当多个规则匹配时，优先级高的规则先执行

添加限流规则
支持精确匹配和正则表达式
设置为0表示不锁定，仅限流
