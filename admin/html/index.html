系统概述

GoWAF V2 是基于 OpenResty + Lua 开发的 Web 应用防火墙（WAF）系统，提供全面的访问控制、流量限制、实时监控和告警功能。系统采用模块化设计，支持多租户架构，可灵活配置各种安全策略。

技术架构

OpenResty Lua MySQL Redis HTML5 CSS3 JavaScript Bootstrap

注意事项

  • 配置生效：修改配置后实时生效，无需重载配置，只有Nginx重启后才需要重新加载配置
  • Nginx重启：如果Nginx进程被杀掉或重启，需要重新加载GoWAF配置，确保WAF功能正常运行
  • 计划任务：需要设置计划任务每秒执行一次告警日志消费接口，将Redis队列中的告警日志写入MySQL数据库
  • 数据保留：历史统计数据保留7天，告警数据保留30天，请及时备份重要数据
  • 内存管理：共享字典大小需要根据实际访问量配置，避免内存溢出
  • Token安全：Token有效期2小时，建议定期更换密码，确保系统安全
  • 性能影响：QPS统计在log_by_lua_file阶段执行，不影响请求响应时间

数据流程

请求处理流程：

  1. Nginx接收请求
  2. access_by_lua_file阶段（main.lua）
  3. 初始化系统（core/init.lua）
  4. 访问控制（access/main.lua）
  5. 业务逻辑处理
  6. log_by_lua_file阶段（log.lua）
  7. 返回响应
